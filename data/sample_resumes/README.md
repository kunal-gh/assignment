# Sample Resume Data

This directory contains sample resume files for testing and demonstration purposes.

## Files Included

- `sample_resume_1.pdf` - Software Engineer with Python/ML background
- `sample_resume_2.pdf` - Data Scientist with R/Statistics experience  
- `sample_resume_3.pdf` - Frontend Developer with React/JavaScript skills
- `sample_job_description.txt` - Example job posting for ML Engineer role

## Usage

These files can be used to:

1. **Test the application** without uploading personal resumes
2. **Demonstrate functionality** during presentations
3. **Validate parsing accuracy** across different resume formats
4. **Benchmark performance** with known data

## Privacy Note

All sample resumes contain fictional information and do not represent real individuals. No personal or sensitive data is included in these files.

## Adding New Samples

When adding new sample files:

1. Ensure all information is fictional
2. Use diverse formats and layouts
3. Include various skill sets and experience levels
4. Test that files parse correctly
5. Update this README with new file descriptions
